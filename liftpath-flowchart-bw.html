Penjelasan Flowchart LiftPath:
1. Entry Point: App Launch → Splash Screen → Authentication Check
2. Authentication Flow: Login/Register → Profile Setup → Main App
3. Main Navigation: Bottom Navigation dengan 4 tab utama
4. Home Tab: Dashboard dengan ringkasan aktivitas dan progress
5. Exercise Program: Program latihan terstruktur dengan 6 kategori otot
6. Calories Calculator: BMR calculator dan tracking kalori harian
7. Profile Management: Data pribadi, statistik tubuh, dan pengaturan
8. Exercise Execution: Tracking set, rep, weight dengan timer dan video guide
